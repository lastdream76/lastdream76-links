# lastdream76 Portfolio

[English](#english) | [Русский](#русский)

---

## English

A minimal static portfolio site showcasing links to GitHub, Modrinth mods, and Discord contact.

### Features

- **Light yellow & dark theme** – Clean, modern design with high contrast
- **Bilingual support** – Toggle between English and Russian with a single click
- **No dependencies** – Pure HTML, CSS, and vanilla JavaScript
- **Responsive design** – Works on desktop and mobile

### Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Flexbox, CSS variables for theming
- **JavaScript (Vanilla)** – Language switching, DOM manipulation
- **SVG icons** – GitHub, Modrinth, Discord logos

### Project Structure

```
portfolio/
├── index.html          # Main page (static, no build required)
├── styles.css          # All styles (variables, responsive, compact)
├── favicon.svg         # Logo icon (appears in browser tab)
├── src/
│   └── index.ts        # TypeScript source (optional, for future builds)
├── dist/               # Compiled JavaScript (if using TypeScript)
├── tsconfig.json       # TypeScript configuration
├── package.json        # Dependencies (typescript devDependency)
└── README.md           # This file
```

### Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers

## Русский

Минималистичный статический сайт портфолио со ссылками на GitHub, моды Modrinth и Discord.

### Возможности

- **Светло-жёлтая & тёмная тема** – Чистый, современный дизайн с высокой контрастностью
- **Двуязычная поддержка** – Переключение между английским и русским одним кликом
- **Без зависимостей** – Чистые HTML, CSS и ванильный JavaScript
- **Адаптивный дизайн** – Работает на настольных и мобильных устройствах

### Технологический стек

- **HTML5** – Семантическая разметка
- **CSS3** – Flexbox, CSS-переменные для темирования
- **JavaScript (Vanilla)** – Переключение языка, манипуляция DOM
- **SVG иконки** – Логотипы GitHub, Modrinth, Discord

### Структура проекта

```
portfolio/
├── index.html          # Главная страница (статичная, сборка не требуется)

├── tsconfig.json       # Конфигурация TypeScript
├── package.json        # Зависимости (typescript как devDependency)
└── README.md           # Этот файл
```
